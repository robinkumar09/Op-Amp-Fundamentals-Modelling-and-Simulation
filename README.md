# Op-Amp-Fundamentals-Modelling-and-Simulation
This document is a reorganized write-up of the Op-Amp modeling and simulation
exercise.Using a VCVS-based op-amp with ±1.8 V rails and a 60 dB open-loop gain,
the work reproduces all original calculations,figures,and theoretical results while
giving the prose a refreshed presentation.Analyses include DC sweeps,AC frequency
response,and transient runs across multiple closed-loop topologies: open-loopc heck,unity buffer, inverting/non-inverting amplifiers, difference amplifier, and an integrator.
